💼 **경력**

- **대한민국 공군 — 공군지능정보체계관리단 AI신기술체계개발대 데이터플랫폼팀** (2025.10 – 2026.11)
  내부 공개 파견 공고에 지원하여 서류 및 면접 전형을 통해 선발되어 부서 이동.
  두 가지 핵심 파이프라인 개발을 주도:
  - **Airwards 이미지 검색 파이프라인**: OpenSearch(BGE-M3 신경망 검색), vLLM, MinIO, Airflow를 활용한 RAG 기반 항공 이미지 검색 서비스. Gemma/Qwen 모델로 이미지 캡셔닝 파이프라인 구축 및 키워드 + kNN 하이브리드 검색 구현.
  - **RAG 평가 파이프라인**: RAGAS, DeepEval(커스텀 G-Eval 메트릭), RAGChecker를 활용한 평가 프레임워크 설계 및 구현. Langfuse를 통해 Qwen3-32B를 Judge 모델로 활용.

- **대한민국 공군 — 통신병** (2025.02 – 2025.10)
  통신 특기병으로 복무하며 운용 지원 업무 수행.

- **먼슬리키친** (2024.07 – 2024.08): QA 인턴 — 내부 플랫폼 소프트웨어/시스템 테스트, 검증 및 이슈 리포팅 참여.

- **이트너스** (2023.01 – 2023.02): 연말 세무 처리 시즌 단기 인턴 — 행정 및 시스템 운영 지원.
